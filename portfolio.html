AutoFUTbuilder
Javascript | Node.js | Python | MongoDB
AutoFUTbuilder is a web application that builds the best possible FIFA Ultimate Team squads based on user needs. In background it solves a constraint satisfaction problem with back-propagation techniques, and give the output to user after solutions found.
